05-套用 Bootstrap Dashboard 版型

Posted on 2021-04-10 In vue Disqus:

步驟備忘

  1. 複製後台版型 CSS、HTML
  2. 增加Dashboard.vue
  3. 新增 Dashboard 路由，需驗證
  4. 拆解 Dashboard HTML，把 Navbar、Sidebar 拆成元件，刪除 main 的內容
  5. 將 Navbar、Sidebar 載入 Dashboard.vue，並放置到正確位置。
  6. 建立巢狀路由 修改完看結果，發現打了網址一直看不到 main 的 123 ，最後才發現因為是巢狀路由，所以網址應該打 /admin/products，而我一直以為打 /admin 就會看到 123。

輸入正確網址後，只會在 F12 代碼看到 123，但頁面上卻沒有顯示，查了半天才知道它的HTML 結構有問題，請見圖：

結構中使用 row 包住 sidebar 跟 main，sidebar 跟 main 都有設定正確的 col ，看起來沒什麼問題，可是 sidebar 設了 class=sidebar ，使側選單 fixed 固定在左方，以致於脫離了 row 的 flex box ，因此 main 就從 row 的最左方開始顯示，而不是排列在 sidebar 之後。

修正的方法就是把側選單的 sidebar class 或是 fixed 屬性拿掉，sidebar 與 main 的排列就會正常，頁面上也看得到 123 了。
